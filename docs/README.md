# Документация к проекту geometric_lib

## 1. Общее описание решения

Проект `geometric_lib` представляет собой простую библиотеку на языке Python для вычисления геометрических характеристик двух фигур:  
- **Круг** — вычисление площади и длины окружности (периметра) по радиусу.  
- **Квадрат** — вычисление площади и периметра по длине стороны.

Библиотека состоит из двух модулей:  
- `circle.py` — содержит функции для работы с кругом.  
- `square.py` — содержит функции для работы с квадратом.  

Проект не использует сторонних зависимостей, кроме стандартной библиотеки Python (`math`).

## 2. Описание каждой функции с примерами вызова

### `circle.area(r)`
**Назначение**: вычисляет площадь круга.  
**Параметр**:  
- `r` — радиус круга (число, больше 0).  
**Возвращаемое значение**: площадь круга (тип `float`).  
**Пример вызова**:
```python
from circle import area
print(area(5))  # Вывод: 78.53981633974483
### `circle.perimeter(r)`
**Назначение**: вычисляет длину окружности (периметр круга).  
**Параметр**:  
- `r` — радиус круга (число, больше 0).  
**Возвращаемое значение**: длина окружности (тип `float`).  
**Пример вызова**:
```python
from circle import perimeter
print(perimeter(3))  # Вывод: 18.84955592153876
```

### `square.area(a)`
**Назначение**: вычисляет площадь квадрата.  
**Параметр**:  
- `a` — длина стороны квадрата (число, больше 0).  
**Возвращаемое значение**: площадь квадрата (тип `int` или `float`, в зависимости от входа).  
**Пример вызова**:
```python
from square import area
print(area(4))  # Вывод: 16
```

### `square.perimeter(a)`
**Назначение**: вычисляет периметр квадрата.  
**Параметр**:  
- `a` — длина стороны квадрата (число, больше 0).  
**Возвращаемое значение**: периметр квадрата (тип `int` или `float`).  
**Пример вызова**:
```python
from square import perimeter
print(perimeter(5))  # Вывод: 20
```
